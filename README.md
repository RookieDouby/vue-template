# vue-template
自己创建一个vue-template, 用于快速展开新项目开发，里边集合一些自己新发现的vue新功能或者新玩法

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
